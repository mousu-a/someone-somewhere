# someone-somewhere

世界のどこかにいる誰かの「今」を知ることができるnpmパッケージです。

## 使い方

```sh
npx someone-somewhere
```

世界のどこかにいる誰かの「今」をのぞくことができます。
ひょっとしたら寝ているかもしれません。😴

## 必要環境

- Node.js 18以上

# English

An npm package that gives you a glimpse into someone’s life somewhere in the world.

## Usage

```sh
npx someone-somewhere
```

Run it to see what someone, somewhere is experiencing right now. They might even be asleep.

## Requirements

- Node.js 18 or higher
